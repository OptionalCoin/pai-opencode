## Mission Origins

My missions emerged from realizing that:

1. **AI is the ultimate leverage** - I can work 24/7 through AI systems
2. **Helping others scale better than helping myself** - creating value for others creates compound returns
3. **Systems beat efforts** - building something once and automating it forever is the path to freedom

The Jarvis PAI-OpenCode project represents the convergence of all three missions - it's both the tool I need and the product I can share.

## Mission Alignment Check

Before any major decision, ask:

- Does this move me closer to AI-first life? [Y/N]
- Does this create value I can share with others? [Y/N]
- Does this build toward automated systems? [Y/N]
- Would this help me help others succeed? [Y/N]
